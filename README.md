Dotfiles

Repositório com minhas dotfiles pessoais para Fish, Tmux, Alacritty e FastFetch. 
Contém configurações, funções, completions etc.

Objetivo  
Manter configurações portáveis e versionadas para recuperar rapidamente meu ambiente de terminal e aparência.

Principais componentes

    Fish — shell interativo e funções personalizadas.

    Tmux — multiplexador de terminal e plugins.

    Alacritty — terminal GPU-accelerated com tema e fontes.

    FastFetch — ferramenta de informação do sistema para prompt/lockscreen.
